# Kirtan Soni Portfolio

Professional portfolio website for Data Engineering, Analytics, and AI/ML roles.

## Project Structure

- `index.html` - main portfolio page
- `style.css` - all styling
- `script.js` - interactions and animations
- `error.html` - custom 404 page
- `images/` - all local assets used by the site
- `Resume.pdf` - resume file

## What Was Updated

- Split inline styles and scripts into `style.css` and `script.js`.
- Synced required image assets into `images/`.
- Fixed `error.html` home button to use a repo-safe path (`./index.html`).

## Run Locally

Open `index.html` directly, or run a local server:

```powershell
# from project root
python -m http.server 8000
```

Then open:

`http://localhost:8000`

## Deployment

Deploy as a static site (GitHub Pages, Netlify, Vercel static, etc.).
Ensure all files remain at the same level as in this repository.
